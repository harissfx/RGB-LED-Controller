# Panduan Kontribusi

Terima kasih telah tertarik untuk berkontribusi pada **RGB LED Controller**! 

## Cara Berkontribusi

### 1. Laporkan Bug
Jika Anda menemukan bug:
- Buka [GitHub Issues](https://github.com/username/RGB-LED-Controller/issues)
- Berikan deskripsi detail tentang bug
- Sertakan langkah-langkah reproduksi
- Jelaskan hasil yang diharapkan

### 2. Usulkan Fitur Baru
Untuk fitur baru:
- Diskusikan di [GitHub Discussions](https://github.com/username/RGB-LED-Controller/discussions)
- Jelaskan use case Anda
- Berikan contoh kode jika memungkinkan

### 3. Pull Request

#### Setup Lokal
```bash
git clone https://github.com/username/RGB-LED-Controller.git
cd RGB-LED-Controller
```

#### Membuat Branch
```bash
git checkout -b feature/YourFeatureName
```

#### Commit Guidelines
```bash
git commit -m "Add: description of your feature"
git commit -m "Fix: description of the bug fix"
git commit -m "Docs: update documentation"
```

#### Push & Create PR
```bash
git push origin feature/YourFeatureName
```

Kemudian buat Pull Request di GitHub dengan deskripsi lengkap.

## Standar Kode

- Gunakan indentasi 2 spasi
- Tambahkan komentar untuk logika kompleks
- Nama fungsi: camelCase
- Nama konstanta: UPPER_CASE
- Uji kode sebelum submit
